# Atividade: Refatoração To-Do com Vue 3

Esta atividade consiste na refatoração de um aplicativo de lista de tarefas (To-Do) de JavaScript puro para **Vue 3**.

## 🎯 Objetivos Alcançados
- Recriação da funcionalidade de adicionar, listar, marcar como concluída e remover tarefas.
- Utilização de diretivas essenciais do Vue 3:
  - `v-for`: Para renderização da lista de tarefas.
  - `v-model`: Para sincronização do campo de entrada.
  - `@click`: Para manipulação de eventos.
- **Sem manipulação manual do DOM** (sem `getElementById`, `createElement`, etc).

## 📁 Estrutura
O projeto está contido em um único arquivo `index.html` que utiliza o Vue 3 via CDN para facilitar a execução e visualização.

## 🚀 Como executar
Basta abrir o arquivo `index.html` em qualquer navegador moderno.
